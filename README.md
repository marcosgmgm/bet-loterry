# Bet Lottery

Project create for tests with camunda and event sourcing

## Technology

- Kotlin
- Maven
- PostgreSQL
- Liquibase
- GrayLog
- Docker Compose
- Camunda
- Event Store

## Installation

- DOCKER COMPOSE

  Ubuntu

  ```
  sudo apt-get install docker-compose
  ```

## Execution

1 - Start docker compose

```
docker-compose up
```

2 - Build project

```
mvn clean install
```

## docker-compose

Start containers

```
docker-compose up
```

Clean and remove containers

```
docker-compose rm
```
